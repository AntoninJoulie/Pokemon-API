# API Pokémon - PKMNAPI
## Catch them all !

All the Pokémon data you've ever wanted: 
* Trainer ( `https://www.url.com/trainer` )
* Pokémon ( `https://www.url.com/pokemon` )
* Region ( `https://www.url.com/region` )

Installation NPM ( Node Packet Manager )
```
    $ npm init
    $ npm i mongoose -s
    $ npm i express -s
    $ npm i body-parser -s
    $ npm i yargs -s

    $ npm install
```
